# ETL Schema Loader

Pipeline ETL desarrollado en Python para validación, transformación y carga controlada de información hacia BigQuery.

## Descripción

Este proyecto surge para reducir errores derivados de la inferencia automática de tipos durante cargas masivas desde archivos CSV.

La solución implementa una capa intermedia de validación y transformación utilizando un esquema externo para controlar estructura, tipos de datos y consistencia antes de realizar la carga.

## Funcionalidades

* Validación estructural mediante esquema externo
* Normalización de texto y tipos numéricos
* Conversión controlada de fechas
* Carga automatizada hacia BigQuery
* Validación previa para reducir errores de procesamiento

## Tecnologías

* Python
* Pandas
* BigQuery
* SQL
* JSON
